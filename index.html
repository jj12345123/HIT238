Here are the quiz answers:

What is semantic HTML?

Semantic HTML is the correct use of HTML. It helps with search engines, screenreaders etc.

How do you import a stylesheet in to a HTML document?

In the HTML file add: link rel="stylesheet" href="styles.css" type="text/css" / within the head tags

How do you import javascript in to a HTML document?

In the HTML file add: script type="text/javascript" src="path-to-javascript-file.js" within the head tags

What meta tag can you use to fix the mobile screen width?

Viewport. For example: meta name="viewport" content="width=600, initial-scale=1"

How do you apply CSS only to certain screen sizes?

You can use media queries to to apply CSS only to certain screen sizes

What does mobile first mean when writing CSS?

Mobile first means that styling is applied to mobile devices first. You write the CSS for mobile devices before writing it for desktop and other devices

How does a browser decide the order to apply CSS?

CSS styles are applied based on specificity. CS applies the style that is most specific to the element

Name three javascript best practices.

1. Use meaningful names
2. Use strong type checks
3. Use functions that only do one thing

How do you run a javascript function when a button is pressed?

You can add an eventlistener, which will trigger the Javascript code when a button is pressed

What is the difference between == and === in javascript?

== is used to match elements, but it ignores the data type, while === is also used to match the element and requires them to be the same data type
